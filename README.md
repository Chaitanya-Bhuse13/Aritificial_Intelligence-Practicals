# Artificial Intelligence Practicals

This repository contains the Artificial Intelligence (AI) practical programs implemented in Java as part of the SPPU (Savitribai Phule Pune University) Computer Engineering curriculum.

## 📚 Practicals Included

|      Practical         |                      Description                          |
|------------------------|-----------------------------------------------------------|
| BFS.java               | Breadth First Search (BFS) algorithm for graph traversal. |
| DFS.java               | Depth First Search (DFS) algorithm for graph traversal.   |
| ElementaryChatbot.java | Simple rule-based chatbot implementation.                 |
| Jobs.java              | Job representation used in scheduling algorithms.         |
| JobScheduling.java     | Job Scheduling algorithm to maximize profit.              |
| NQueens.java           | Solution to the N-Queens problem using Backtracking.      |

## 🛠️ Technologies Used

- Java
- Eclipse IDE

## ▶️ How to Run

1. Clone the repository.
2. Open the project in Eclipse or any Java IDE.
3. Navigate to the desired Java file.
4. Run the program as a Java Application.

## 🎯 Purpose

This repository serves as a collection of Artificial Intelligence laboratory practicals for learning and understanding fundamental AI algorithms and concepts prescribed by the SPPU Computer Engineering syllabus.
